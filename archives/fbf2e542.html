题目答案

  • 晏殊有词云：“兔走乌飞不住，人生几度三台”此处“三台”指的是？
    三台，这里指高位，本指星名，《晋书·天文志上》：“三台六星……一曰天柱，三公之位也。在人曰三公，在天曰三台，主开德宣符也

  • 五羖大夫指的是？该典故出自哪里？

百里奚 【出自】《史记·秦本纪》：“乃使人谓楚曰：‘吾媵臣百里奚在焉，请以五羖羊皮赎之。’……缪公大说，授之国政，号早曰‘五羖大夫

  • 据《史记》中记载，黄帝崩，高阳立，高阳崩而高辛立。请问按照现在的亲戚关系来解释，高阳和高辛之间是什么关系？
    堂叔侄关系。　帝颛顼高阳者，黄帝之孙而昌意之子也。　帝喾高辛者，黄帝之曾孙也。高辛父曰蟜极，蟜极父曰玄嚣，玄嚣父曰黄帝。

  • 中国古代神话素来有龙生九子的说法，下列龙子中，传说好负重，力大无穷的是(B)
    A 狻猊 B 赑屃 C 睚眦 D 蒲牢

  • 《论语 微子》中有语说“周有八士”下列属于八士的是(ABD)
    A伯达﹑伯适 B仲突﹑仲忽 C叔向﹑叔夏 B、季随﹑季騧

  • 茶是中国人民的传统饮料，具有悠久的历史，下列属于发酵茶的是(ACDE)
    A 红茶 B绿茶 C 乌龙茶 D 黄茶 E 砖茶 F 白茶

  • 苏东坡在在一诗中写到“根茎与花实，收拾无弃物”该句说的是一种名贵中药材的名称，请说出该中药的名称？
    枸杞

  • 麝香和灵猫香是我国两种著名的香料，请问麝香和灵猫香产香的两种动物的性别分别是？
    雄麝和雌灵猫

  • 在古诗文中，不同颜色的马通常用不同的字表示，请说出下面几句诗文中表示马的字所代指的马的颜色？

    • 踯躅青骢马，流苏金镂鞍。——《玉台新咏·古诗为焦仲卿妻作》 ——骢
    • 骝马新跨白玉鞍，战罢沙场月色寒——《军行》——李白——骝
    • 四骐翼翼。——《诗·小雅·采芑》 ——骐
    • 骅骝开道路,鹰隼出风尘。――杜甫《奉简高三十五使君》——骅

骝——黑鬣、黑尾巴的红马
骢——青白杂毛的马。今名菊花青马
骐——有青黑色纹理的马
骅——古代指赤色的骏马

  • 《周礼》中提及“以玉作为器，以礼天地四方。”其中用于“礼天”和“礼地”的分别是？(AB)
    A.璧 B.璋 B.琮 C.珪 D.璜 E.琥

  • 请说出下列玉器的作用：

    • 玉琀
    • 玉玦
    • 玉衣

玉衣是指包裹全身从头到脚每一个部位的衣罩，其外观与真人的体形相同，专为罩尸之用
玉琀又称“饭含”，是放在死者口中的玉器。
玦是指一种环形而有缺口的玉器，是最古老的玉制装饰品之一。在古代主要是被用作耳饰和佩饰

  • 唐三彩是盛行于唐代的一种陶器，请问唐三彩的基本釉色是哪三种？
    黄、褐、绿

  • 三字经中有“匏土革 木石金 丝与竹 乃八音”之言请问这里八音的划分是依据是什么？
    根据制作乐器的材料来划分

  • 根据上题中的八音划分，给“铙，埙 竽，琴”四种乐器分类，按顺序写出类别
    金，土，匏，丝

  • “五花八门”原指“五花阵”和“八门阵”后来又用来比喻各行职业的暗语，下列对应正确的是(BCD)
    A 金菊花——比喻上街为人治病的郎中 木棉花——比喻卖茶的女人
    B 水仙花——比喻酒楼上的歌女 火棘花——比喻玩杂耍的
    C 一门巾——算命占卦 二门皮——卖草药的
    D 五门评——说评论弹者 六门团——街头卖唱者

详细释义
　　五花：
　　金菊花——比喻卖茶的女人。
　　木棉花——比喻上街为人治病的郎中。
　　水仙花——比喻酒楼上的歌女。
　　火棘花——比喻玩杂耍的。
　　土牛花——比喻一些挑夫。
　　八门：
　　一门巾——算命占卦。
　　二门皮——卖草药的。
　　三门彩——变戏法的。
　　四门挂——江湖卖艺的。
　　五门评——说评论弹者。
　　六门团——街头卖唱者。
　　七门调——搭棚扎纸的。
　　八门聊——高台唱戏的。
　　简称巾、皮、彩、挂、平、团、调、聊，是谓“八门”。

  • 酒是中国古代文化不可分割的一部分，在”岁盏后推蓝尾酒，春盘先劝胶牙饧”该句中提到的“蓝尾酒”的意思是？
    唐代饮宴时，轮流斟饮，至末坐，称”蓝尾酒”

  • 古人一起饮酒时，次序非常重要，请选出下面一种饮酒顺序与其他三种不同的( C )
    A 松醪酒 B 竹叶青 C 屠苏酒 新丰酒后饮屠苏。”

　一般人饮酒，总是从年长者饮起；但是饮屠苏酒却正好相反，是从最年少的饮起。也就是说合家欢聚喝饮屠苏酒时，先从年少的小儿开始，年纪较长的在后，逐人饮少许。宋朝文学家苏辙的《除日》诗道：“年年最后饮屠苏，不觉年来七十余。”说的就是这种风俗。有人不明白这种习惯的意义，董勋解释说：“少者得岁，故贺之；老者失岁，故罚之。”这种风俗在宋朝仍很盛行，如苏轼在《除夜野宿常州城外》诗中说：“但把穷愁博长健，不辞最后饮屠苏。”
