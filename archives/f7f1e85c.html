2014总结——记下流水账

发表于 2014-12-20 更新于 2020-11-18 分类于 闲情 阅读次数：
繁/简：繁体
本文字数： 2.4k 阅读时长 ≈ 2 分钟

在每年的进行年终总结时总是在心里一遍遍谨慎的措辞，总奢望行文可以对过去的一年做出最为精当的概括。但即使每次删删改改好几次，写出来的文字总是差强人意。最后也只能是信马由缰，写到哪算哪！

如果本来不读博的话，年初我本该跟着刘宇，坤琦一起毕业，走向工作岗位了。出于内心的执念，还是坚持了内心的想法，即使想想现在的处境有点挣扎，但从不曾有丝毫后悔过！2014年生活上经历的第一件事大概就是硕士同学的毕业，当然这里的同学包括实验室的基友。自然，我不是个怀旧的人，从小到大的毕业散伙饭我很少有悲伤的情绪。对于他们的毕业值得高兴的一点是都找到了不错的工作，据了解目前的生活状态都不错；不爽的一点是，每次聊天开黑的时候，总是会被问到“你什么时候毕业，我们到时带着孩子回来看你”。对于一群损友来说，戳戳痛处，互黑一下就是增进感情的最好的途径了。刘宇毕业后，吃外卖打三国杀的时候再也没有观众了;坤琦毕业了，也没人一起讨论海贼王的情报了；宿舍的基友们毕业了每次回宿舍都空荡荡了，再也没人开捡香皂的玩笑了；打三国杀的基友们毕业了，组局总是缺人。在三月份过后，经历过一段艰难的日子，经济危机，感情危机接踵而来，有点应接不暇的感觉，好在处理的还算从容，至少表面上没人看出太过狼狈。也慢慢学会了找些兼职做，慢慢的计划接触了社会。这段挣扎的日子，更加深刻的认识了自己，从心态上来讲变得更加平和，宽容。现在想想还应该感谢这段经历！

也正是在上半年这段时间，选修了高老师的课，对高老师有了进一步的了解。深深的为高老师身上所散发的人格魅力所震撼，也为自己能遇见这么高贵的老师而庆幸。我第一次从人身上感受到了灵魂的高贵，在此之前我一直以为所有人的灵魂都是平等的，在老师身上第一次发现了有些人的灵魂确实要更加高尚，这种高尚与地位无关，而是出于对知识的敬畏；自由意志的追寻；对生命的尊重。认识高老师让我再次梳理了自己灵魂上的污渍。这算是今年上半年最大的收获之一了。

暑假还是和原来一样待在学校没回家，科研课题就像个死胎，怎么憋也感觉进展不大，不过怎么说也是自己的孩子。试了很多方法，半途而废了很多次这是最后悔的，原来一直引以为豪的韧性在这段时间也被消磨殆尽。实验室小组里的事也被我弄的一塌糊涂，好几次都有焦头烂额的感觉，说到这里还得感谢一下师弟师妹们的包容。无数次尝试改掉注意力不集中，做事拖拉的毛病，最后都没能成功，这算是本年度最大的讽刺事件了。中间还在Coursera上报了好几门课程，但真正拿到证书的只有一门最为简单的，其他的都因为各种原因中断。断断续续的一直憋到了10月下旬，开始赶实验室的项目，我负责的事情倒不算太多，就是比较琐碎。整个下半年每个周一上午都感觉头被各种琐事撕碎了一样，总会有事落下。期间接触了Spring MVC, JavaScript, PHP, Python 等一系列各种技术，代码渣的形象彻底暴露，写几行代码就跟便秘似的总是出不来，出来也是各种 bug。在做工程的过程中，发现了解业界的重要性，一直待在学校的局限性渐渐的局限了我的视野。在我小时，觉得最大的局限在知识，外界条件的限制上，大了才发现最大的局限来自于自身，自身的眼界和气度最直观的就是影响到自身做出的各种抉择。其他诸如平时的品性，小肚鸡肠，缺乏包容之心也在这中间能够得到体现。

在年初的时候定下了阅读完一百本课外书，写下十万字的计划。到上个星期一总结，满打满算才70本左右。写过的东西如果将所有杂七杂八的加起来，估计也够十万字，但很多文字其实是没有质量保证的。写书评的习惯，从八月份开始坚持到现在大约四个月左右，中间有过推迟，但好在一直没有中断，一共得有二三十篇了吧，加上去年的可能有接近50篇了。按照原计划，写够256篇的时候，就自己出一版电子书，这是过去的一年中值得欣慰的事吧。最近两年都在如饥似渴的抓紧阅读与联系写作，当然我并不是想成为一个作家或者是文字工作者，只是在目前的自我认识中个人知识缺乏，目光浅短，思维也缺少清晰的逻辑而通过阅读和写作来提高是当前所能找到的最为廉价和有效的途径。从写东西上面，我缺乏灵性，即使写的再流畅的文字中间也不会带着一种灵动，我见过很多文字中夹着着天然灵动天赋的人，我想我经过努力总能写出四平八稳的东西，那样也足够了。

下半年在业余时间还做了很多事，弄了阅读群，阅读公共号，最近还和人一起弄了阅读论坛(这个主要是网友老毛弄的，我沾光)，并且托大家的福，把人数拉起来了，首功之人当属高老师。面临的一个问题是阅读气氛还是没起来，任重而道远。今年通过业余时间总算实现了经济上的自给自足了，买书的欲望暂时得到了释放，明年再接再厉，争取早日把想买的书都买齐。其他还杂七杂八事还做了一堆，不过都不能写到总结里！嗯…我还是写到日记里吧。

过去的一年还认识了很多新朋友，先是同楼一块刷牙的哥们，目前还只喝过一次酒，却也觉得特别投缘。还有一起玩LateX的几个朋友，说到这里颇为抱歉的是答应ddd两周一篇LateX技术文章，结果一学期就弄了两篇还是不伦不类的。这里还是要多亏了勤劳的L帮我们撑起了后期的场面，才没导致我们的网站荒芜。还认识了很多书友，还有一起上英语课n3q。最后还有实验室最近来得很多师弟师妹们，人数太多我已经看得眼花缭乱。当然，还有一个神秘人物，我就写在日记里吧！

以上就差不多是整年的琐事，杂七杂八的吧。需要反省的事还包括下半年动过一次肝火，现在回想起来很不应该，可以变得更有涵养的！提出检讨。最后，希望明年一切顺利，预想中的事情都能顺利实现，身边的亲人朋友都能身体健康。前两年每年总结最后都会怀念朋友的话题，今年就不扯这些虚的了，我2月8号回大湖南，有想要请客吃饭的朋友赶紧Mark,要聚会玩耍的朋友们也赶紧早日组织，我已经练习了多次“年轻的朋友们，我们来相会”！
